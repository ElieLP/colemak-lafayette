Colemak-Lafayette
================================================================================

This project is a fork of the [qwerty-lafayette](https://github.com/fabi1cazenave/qwerty-lafayette) layout.
Same principle as qwerty-lafayette but for colemak layout, with tarmack intermediary steps.
